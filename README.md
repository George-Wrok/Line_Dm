# LINE DM Pro 使用手冊

本工具幫助您快速將促銷文案與圖片轉傳給多位 LINE 好友。

## ✨ 強大功能
- **雙模式編輯**：提供「標準」與「Flex JSON」模式，滿足快速群發與專業排版需求。
- **響應式設計**：完美適配手機與電腦操作，出門在外也能輕鬆發送。
- **無縫整合**：支援 [LINE Flex Message Simulator](https://developers.line.biz/flex-simulator/)，設計完成後貼上 JSON 即可。
- **自動驗證**：JSON 格式即時檢測，確保發送內容無誤。

## 操作流程
1. **選擇模式**：在頂部切換「標準模式」或「Flex JSON 模式」。
2. **編輯內容**：
   - 標準模式：輸入文字、拖曳或點選上傳圖片。
   - Flex 模式：貼上 Flex Simulator 產生的 JSON 代碼。
3. **預覽確認**：在右側（或下方）的手機模型中檢視訊息預覽。
4. **開始群發**：點擊按鈕，勾選多位好友後送出。
